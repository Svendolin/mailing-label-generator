1) CREATE TABLE Query-SQL Befehl um die Datenbank zu speisen:
 - Vier Spalten
 - users_id = ID, die jeder nutzer kriegt, um 1 steigend (AUTO_INCREMENT)
 - users_uid = Nutzername
 - users_pwd = Passwort
 - users_email = Nutzeremail:

CREATE TABLE users (
  users_id int(11) AUTO_INCREMENT PRIMARY KEY not null,
  users_uid TINYTEXT not null,
  users_pwd LONGTEXT not null,
  users_email TINYTEXT not null
)
